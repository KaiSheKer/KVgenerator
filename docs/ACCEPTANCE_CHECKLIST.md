## 🐛 常见问题排查

### 问题1: 提示词完全相同

**检查**:
```javascript
// 在浏览器控制台运行
document.querySelector('textarea').value
```

**解决**:
- 硬刷新浏览器（Cmd+Shift+R）
- 清除浏览器缓存
- 检查`promptGeneratorV2.ts`是否正确保存

### 问题2: 图像生成失败

**检查**:
1. 查看浏览器控制台（F12）
2. 查看Network标签，找到`/api/generate`请求
3. 查看Response内容

**常见错误**:
```
错误: GEMINI_API_KEY not configured
解决: 检查.env.local文件

错误: Model unavailable
解决: 检查模型名称是否正确

错误: 生成失败 (500)
解决: 查看服务器日志
```

### 问题3: V2效果很差

**可能原因**:
1. 产品图质量太高，两种方式都能很好还原
2. 选择的产品类型不适合V2的精简方式
3. 模型的随机性导致单次结果不佳

**建议**:
- 生成多张对比（2-3张）
- 尝试不同的产品类型
- 尝试不同的风格配置
