## 🐛 常见问题

### Q1: 图像生成很慢怎么办？

**A**: Pro版本需要10-15秒，这是正常的。质量比速度更重要。

### Q2: 控制台显示aspect ratio错误？

**A**: 检查配置：
```bash
cat .env.local | grep GEMINI_IMAGE_MODEL
# 应该输出: gemini-3-pro-image-preview
```

然后重启服务器：
```bash
lsof -ti:3000 | xargs kill -9
npm run dev
```

### Q3: V2和V1提示词一样？

**A**: 硬刷新浏览器（Cmd+Shift+R）清除缓存。

### Q4: 生成的图像质量差？

**A**: Pro版本已经是最高质量了。如果还差，可能是：
- 产品图质量问题
- 提示词需要优化
- 模型的随机性（生成多张对比）
