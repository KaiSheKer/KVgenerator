# 情人节时光回忆录 - 设计文档

**日期：** 2025-02-14
**项目名称：** Valentine Memory Book - 复古胶片时光相册
**目标用户：** 女朋友（情人节礼物）
**部署方式：** Vercel托管 + HTTPS链接分享

---

## 项目概述

一个复古胶片风格的照片回忆录网站，通过翻页相册的形式展示10-20张精选照片，记录两人的美好瞬间。整体设计追求温暖、怀旧的情感氛围，让每张照片都能讲述一个故事。

---

## 核心设计元素

### 视觉风格定位
- **整体风格：** 90年代胶片摄影风格，温暖怀旧
- **情感基调：** 浪漫、用心、细节体现
- **核心价值：** "这些细节我都记得"

### 配色方案
- **背景色：** 米白色 `#f5f0e8` - 模拟老相册纸张
- **文字色：** 深棕色 `#3d342b` - 温暖易读
- **照片偏色：** 暖色调轻微偏移（模仿胶片冲洗）
- **装饰色：** 红色印章 `#c94a4a` - 日期印记

### 字体选择
- **标题字体：** 手写风格中文手写体（Google Fonts: "Ma Shan Zheng" 或 "Zhi Mang Xing"）
- **正文字体：** 宋体/Georgia 衬线字体 - 书写的温度感
- **数字字体：** 等宽字体 - 胶片日期印记

### 视觉特效
- **胶片噪点：** 轻微颗粒噪点覆盖全屏
- **暗角效果：** 四周轻微渐暗
- **照片边框：** 白色边框（8-12px）+ 胶卷齿孔效果
- **照片阴影：** 柔和投影（增加立体感）
- **随机旋转：** 每张照片轻微旋转（-3°到+3°）
- **纸张纹理：** 背景细微纹理，模拟相册纸张

- **居中对齐：** 所有元素垂直水平居中
- **留白充足：** 给内容呼吸空间，不拥挤

## 内容组织方式

### CSV配置文件（用户编辑）

**文件位置：** `public/photos.csv`

**格式：**

| image | date | location | title | story |
|-------|------|----------|-------|--------|
| photo-001.jpg | 2024.03.15 | 星巴克 | 那个下午你说 | 还记得我们在咖啡馆坐了整个下午... |
| photo-002.jpg | 2024.05.20 | 外滩 | 第一次看夜景 | 那天风很大，你帮我整理围巾... |

**字段说明：**
- `image`: 照片文件名（对应 `public/photos/` 文件夹）
- `date`: 日期（格式：YYYY.MM.DD）
- `location`: 地点（可选，用于印章显示或未来扩展）
- `title`: 短标题（手写体显示）
- `story`: 故事文字（多行文字，支持换行）

### 文件结构

```
public/
├── photos/
│   ├── photo-001.jpg
│   ├── photo-002.jpg
│   └── ...
├── background-music.mp3    # River Flows In You - Yiruma
└── photos.csv              # 内容配置文件
```

### 照片处理要求
- **格式：** JPG/PNG（iPhone HEIC需转换）
- **尺寸：** 建议1920x1080或更大（自动适配）
- **命名：** photo-001.jpg, photo-002.jpg 顺序命名
- **质量：** 70-80%压缩比（平衡质量和加载速度）

## 技术方案

### 技术栈
- **框架：** Next.js 15（React框架）
- **样式：** Tailwind CSS（快速开发 + 响应式）
- **部署：** Vercel（零配置 + HTTPS + 全球CDN）
- **字体：** Google Fonts（在线加载）
- **图标：** Lucide React（轻量级图标库）

### 关键功能实现

#### 1. CSV数据解析
```javascript
// 使用 Papa Parse 库解析CSV
import Papa from 'papaparse';

// 读取并转换CSV为JSON数组
const photos = await parseCSV('/photos.csv');
```

#### 2. 胶片效果CSS
```css
.film-effect {
  /* 胶片噪点 */
  background-image: url('data:image/svg+xml;...');
  filter: contrast(1.1) saturate(0.9);
}

.photo-frame {
  /* 胶卷齿孔边框 */
  border-left: 20px solid #1a1a1a;
  background: repeating-linear-gradient(...);
}
```

#### 3. 音乐自动播放
```javascript
// 低音量淡入
audio.volume = 0;
audio.play();
fadeInAudio(audio, 0.4); // 渐变到40%音量
```

#### 4. 键盘导航
```javascript
useEffect(() => {
  const handleKeyPress = (e) => {
    if (e.key === 'ArrowLeft') goToPrev();
    if (e.key === 'ArrowRight') goToNext();
  };
  window.addEventListener('keydown', handleKeyPress);
  return () => window.removeEventListener('keydown', handleKeyPress);
}, []);
```

### 响应式设计
- **移动端（<768px）：** 单列布局，照片占宽度70-80%
- **平板端（768px-1024px）：** 单列布局，照片占宽度60%
- **桌面端（>1024px）：** 单列布局，最大宽度800px居中

### 性能优化
- **照片懒加载：** 使用 Next.js Image 组件
- **音乐预加载：** 页面加载时预加载音频
- **CSV缓存：** 客户端缓存解析结果
- **字体优化：** display=swap，首屏快速显示

## 设计原则

### YAGNI（You Aren't Gonna Need It）
- 专注核心功能，不过度设计
- 不添加炫技但无用的特效
- 不为了技术而技术

### 情感优先
- 每个设计决策都要问"这样更浪漫吗？"
- 简洁 > 复杂，温暖 > 华丽
- 细节体现用心，不是靠堆砌元素

### 可维护性
- 代码清晰，注释充分
- 用户可编辑内容（CSV文件）
- 易于部署和更新
