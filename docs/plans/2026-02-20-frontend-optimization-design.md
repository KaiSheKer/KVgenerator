## 设计目标

### 核心目标

- **视觉美观度**: 使用暖色渐变、大圆角、柔和阴影营造现代感
- **用户体验**: 极简布局,聚焦核心操作,清晰的操作流程
- **品牌感**: 专业且友好,适合作品集展示的 AI 工具风格

### 设计原则

1. **极简至上**: 去除所有无关元素,只保留核心操作
2. **视觉引导**: 用渐变色、大圆角、阴影引导用户视线
3. **即时反馈**: 悬浮、点击都有视觉反馈
4. **温暖商业**: 橙粉渐变营造亲和力,不失专业感

## 全局设计系统

### 配色方案

```css
/* 主色系 - 暖色渐变 */
--primary-gradient: linear-gradient(135deg, #FF6B6B 0%, #FF8E53 100%);
--accent-gradient: linear-gradient(135deg, #FFD93D 0%, #FFAA4C 100%);

/* 中性色 */
--background: #FFF5F5;          /* 极浅粉背景 */
--surface: #FFFFFF;             /* 纯白卡片 */
--border: #FFE5E5;             /* 浅粉边框 */

/* 文字色 */
--text-primary: #2D3748;       /* 深灰 */
--text-secondary: #718096;     /* 中灰 */
--text-tertiary: #A0AEC0;      /* 浅灰 */

/* 功能色 */
--success: #48BB78;            /* 绿 */
--error: #F56565;              /* 红 */
--warning: #FFD93D;            /* 黄 */
```

### 圆角系统

- **卡片**: `2xl` (24px)
- **按钮**: `xl` (12px)
- **输入框**: `lg` (8px)
- **统一原则**: 所有元素使用大圆角,营造柔和感

### 阴影系统

```css
/* 卡片阴影 */
--shadow-sm: 0 2px 8px rgba(255, 107, 107, 0.08);
--shadow-md: 0 4px 16px rgba(255, 107, 107, 0.12);
--shadow-lg: 0 8px 32px rgba(255, 107, 107, 0.16);

/* 悬浮阴影 */
--shadow-float: 0 12px 48px rgba(255, 107, 107, 0.20);
```

### 动画系统

**过渡时间**:
- 快速: 150ms (按钮点击)
- 标准: 300ms (悬浮效果)
- 慢速: 500ms (页面切换)

**缓动函数**:
- 标准: `cubic-bezier(0.4, 0, 0.2, 1)`

## 页面设计

### 1. 首页(极简版)

**布局结构**:
```
┌─────────────────────────────────────────────┐
│                                             │
│          [巨大的上传卡片区域]                │
│                                             │
│           📸                                │
│     拖拽产品图片到此处                        │
│        或点击上传                            │
│                                             │
│   支持 JPG、PNG、WEBP,最大 10MB             │
│                                             │
│                                             │
│          [开始分析 → 按钮]                   │
│          (上传后显示)                        │
│                                             │
```

**设计要点**:
- 上传区域占据屏幕 60% 高度,居中显示

**交互状态**:
- 默认: 虚线边框
- 悬浮: 实线边框 + 渐变色 + 上浮效果
- 拖拽: 边框发光 + 背景变色

## 交互设计

### 微动效细节

**按钮交互**:
```
正常 → 悬浮 → 点击
阴影-md → 阴影-lg → 缩小 95%
无上浮 → 上浮 4px → 按下效果
```

**卡片交互**:
```
正常 → 悬浮 → 点击
阴影-md → 阴影-lg → 上浮 6px
透明度 1 → 0.98 → 边框高亮
```

**进度条动画**:
- 条纹动画(striped) + 渐变色
- 条纹从左到右流动,2s 一圈
- 完成时粒子喷射效果

**页面切换**:
- 淡入淡出 + 轻微上滑
- 过渡时间: 300ms
- 使用 Framer Motion 或 CSS transition

## 响应式设计

### 断点

- **Desktop**: > 1024px (4列网格)
- **Tablet**: 768px - 1024px (2-3列)
- **Mobile**: < 768px (1列,大尺寸按钮)

### 移动端优化

**首页**:
- 上传卡片占据屏幕 80% 高度
- 图标和文字放大

**风格选择**:
- 改为横向滚动
- 卡片尺寸增大
- 触摸时显示操作按钮

**通用优化**:
- 按钮高度增至 48px
- 字号适度放大
- 间距增加,避免误触

## 性能和可访问性

### 性能优化

- **图片懒加载**: 使用 Intersection Observer
- **代码分割**: 按路由分割
- **动画优化**: 使用 transform 和 opacity
- **骨架屏**: 加载时显示占位符

### 可访问性

- **键盘导航**: 所有交互支持 Tab 键
- **ARIA 标签**: 完整的无障碍属性
- **焦点指示**: 清晰的焦点环(渐变色)
- **颜色对比**: 符合 WCAG AA 标准(4.5:1)
