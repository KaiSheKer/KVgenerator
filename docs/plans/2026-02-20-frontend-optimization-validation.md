## 2. 手工验证步骤

### 2.1 首页与上传

1. 打开首页，确认出现「左侧控制区 + 右侧预览区」双栏布局  
2. 拖拽上传 JPG/PNG/WEBP（<=10MB）验证成功  
3. 上传非图片或超 10MB 文件，确认错误提示

### 2.2 分析与编辑

1. 点击「开始分析」，确认加载态显示且可跳转编辑页  
2. 在编辑页修改品牌名/卖点/颜色，确认输入可用且状态保留  
3. 点击下一步，确认进入风格页

### 2.3 风格与提示词

1. 风格卡片 hover 与 selected 态是否明显  
2. 推荐标签显示正确  
3. 提示词页顶部切换按钮能切换内容  
4. 复制中文提示词按钮可用

### 2.4 生成与画廊

1. 点击生成，确认进度条、状态卡片、当前任务提示实时更新  
2. 若设置 `NEXT_PUBLIC_GENERATION_LIMIT`，确认生成张数受控  
3. 生成完成后进入画廊，确认预览弹层与单图下载可用  
4. “下载全部”文案与行为一致（逐张下载，不标 ZIP）

### 2.5 响应式

1. 桌面宽屏（>=1280）确认双栏完整  
2. 平板宽度（768~1279）确认布局不溢出  
3. 手机宽度（<768）确认卡片堆叠与按钮可点击

## 3. 验收标准

1. 视觉风格对齐参考图方向（深色霓虹、渐变按钮、工作台分区）  
2. 全业务链路可完成（上传 -> 生成 -> 下载）  
3. 自动化检查全部通过
