# KV 无字生图 + 前端叠字验证文档

## 1. 验证目标

1. 模型阶段不再渲染营销文案（仅保留包装原生印刷）。
2. 前端叠字可稳定输出可读中文。
3. 画廊详情可切换“成品图/AI原图”。
4. 叠字失败时可自动回退原图，不影响主流程。

---

## 2. 静态验证

执行：

```bash
npm run lint
npm run typecheck
npm run build
```

预期：
1. 三项全部通过。

---

## 3. 功能验证步骤

### 3.1 基础流程

1. 启动：`npm run dev` 或 `npm start`。
2. 上传产品图 -> 分析 -> 风格 -> 提示词 -> 生成。
3. 进入画廊，确认海报正常展示。

### 3.2 无字生图验证

1. 打开任一海报详情。
2. 点击 `AI原图` 切换按钮。
3. 观察画面中是否出现额外营销标题/CTA 文本。

预期：
1. AI 原图不包含新增营销文案。
2. 包装本身印刷内容正常保留。

### 3.3 前端叠字验证

1. 详情页切回 `成品图`。
2. 检查中文标题、副标题、卖点和 CTA 是否清晰。
3. 连续生成两次同一海报，比较文字布局稳定性。

预期：
1. 中文不乱码。
2. 排版位置和层级基本一致。

### 3.4 回退容错验证

在 mock 模式或受限网络下触发叠字异常时检查：
1. 海报仍可生成并展示。
2. 角标显示 `原图`（若 overlay 未应用）。

---

## 4. 验收标准

1. 生成链路完成 `raw -> final` 合成。
2. 默认下载/展示为成品图。
3. 详情支持成品与原图切换。
4. 构建与运行均稳定。
