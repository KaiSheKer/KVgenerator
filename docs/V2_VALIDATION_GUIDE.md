# V2 提示词验证指南

## 🎯 验证目标

验证新版提示词生成器（V2）是否比现有版本（V1）产生更好的出图效果。

---

## 📋 验证步骤

### 步骤1：启动项目

```bash
cd /Users/ericcao/CascadeProjects/kv-generator
npm run dev
```

访问: http://localhost:3000

---

### 步骤2：上传产品并选择风格

1. **上传产品图**
   - 选择一张清晰的产品图
   - 建议使用橙子、饮料包装等

2. **等待AI分析**
   - 系统会自动提取产品信息
   - 检查提取的品牌、卖点、色彩是否准确

3. **选择视觉风格**
   - 建议选择 `自然有机` 或 `杂志编辑`
   - 选择排版风格（如：粗衬线）
   - 选择画面比例（9:16竖版）

4. **点击"下一步"**

---

### 步骤3：进入对比测试页面

1. **点击"🧪 V1 vs V2 对比测试"按钮**
   - 在提示词页面顶部，有两个按钮：
     - `← 上一步`
     - `🧪 V1 vs V2 对比测试` ← 点击这个

2. **等待对比数据生成**
   - 页面会自动生成V1和V2的提示词
   - 显示统计对比（字符数、压缩比）

### 步骤4：查看提示词对比


**预期结果：**
- V1: 600-1200字符

- 压缩比: 40-60%

#### 📝 提示词内容对比

| 维度 | V1 特征 | V2 特征 |
|------|---------|---------|
| **结构** | 模块化（**产品还原**、**品牌信息**...） | 叙述式（段落流） |
| **卖点数量** | 5个全部展示 | 最多2个核心卖点 |
| **可读性** | 需要跳行阅读 | 一次性读完 |
| **视觉描述** | 需求文档风格 | 画面描述风格 |

### 步骤5：生成图像对比

1. **点击"生成图像 (V1)"按钮**
   - 使用V1提示词生成图像
   - 等待生成完成（约5-10秒）
   - 图像会显示在下方"V1 生成的图像"区域

2. **点击"生成图像 (V2)"按钮**
   - 使用V2提示词生成图像
   - 等待生成完成
   - 图像会显示在"V2 生成的图像"区域

3. **对比生成的图像**

#### 🔍 图像质量评估维度

从以下5个维度对比V1和V2生成的图像：

##### 1️⃣ 产品还原度 ⭐⭐⭐⭐⭐

**评估标准：**
- ⭐⭐⭐⭐⭐: 完美还原，包装、颜色、Logo、文字与原图一致
- ⭐⭐⭐⭐: 高度还原，细节有轻微差异
- ⭐⭐⭐: 基本还原，但有明显差异
- ⭐⭐: 还原度较低，关键元素缺失
- ⭐: 完全不还原

**观察要点：**
- 包装设计是否保持一致？
- 颜色是否准确？
- Logo位置和拼写是否正确？
- 产品形状和比例是否正确？

##### 2️⃣ 视觉美感 ⭐⭐⭐⭐⭐

**评估标准：**
- ⭐⭐⭐⭐⭐: 专业级大片质感，构图完美
- ⭐⭐⭐⭐: 高质量，构图优美
- ⭐⭐⭐: 质量一般，构图尚可
- ⭐⭐: 质量较低，构图混乱
- ⭐: 非常粗糙

**观察要点：**
- 光影是否自然？
- 构图是否平衡？
- 色彩是否和谐？
- 整体是否有高级感？

##### 3️⃣ 风格一致性 ⭐⭐⭐⭐⭐

**评估标准：**
- ⭐⭐⭐⭐⭐: 完全符合所选风格（如：自然有机）
- ⭐⭐⭐⭐: 基本符合风格，有轻微偏差
- ⭐⭐⭐: 风格不够明确
- ⭐⭐: 风格混乱
- ⭐: 完全不符合风格

**观察要点（以"自然有机"为例）：**
- 是否有自然元素（植物、大地色系）？
- 背景是否简洁？
- 光影是否柔和自然？
- **关键检查：** 是否出现矛盾元素（如霓虹灯、赛博效果）？

##### 4️⃣ 文字可读性 ⭐⭐⭐⭐⭐

**评估标准：**
- ⭐⭐⭐⭐⭐: 文字清晰易读，字体大小合适
- ⭐⭐⭐⭐: 文字基本可读，有轻微模糊
- ⭐⭐⭐: 文字较小或部分模糊
- ⭐⭐: 文字难以辨认
- ⭐: 无文字或文字乱码

**观察要点：**
- 品牌名称是否清晰？
- 主标题是否醒目？
- 中英文是否都正确显示？
- 文字是否被遮挡或模糊？

##### 5️⃣ 整体质量 ⭐⭐⭐⭐⭐

**综合评估：**
- ⭐⭐⭐⭐⭐: 可以直接用于商业投放
- ⭐⭐⭐⭐: 需要轻微调整后使用
- ⭐⭐⭐: 需要较大修改
- ⭐⭐: 仅作为参考
- ⭐: 完全不可用

## ❓ 常见问题

### Q1: 生成的图像差异不明显？

**A:** 可能的原因：
1. 产品图本身质量很高，两种方式都能很好还原
2. 选择的产品类型不适合V2的精简方式
3. 图像生成模型对提示词差异不敏感

**建议：**
- 尝试不同的产品类型（包装复杂的商品）
- 尝试不同的视觉风格
- 生成多张图像对比

### Q2: V2的文字生成效果差？

**A:** V2专注于视觉描述，文字生成依赖于图像模型的能力。

**建议：**
- 如果需要高质量文字，考虑使用Overlay方式后期添加
- 或者使用V1的完整提示词

### Q3: 如何判断风格一致性？

**A:** 检查是否有矛盾元素：
- 选择"自然有机"风格
- 检查是否出现霓虹灯、赛博效果
- 检查色彩是否符合自然风格（大地色、绿色）
- 检查背景是否简洁自然
