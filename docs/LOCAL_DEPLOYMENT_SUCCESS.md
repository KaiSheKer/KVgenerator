## 🚀 快速开始测试

### 方式1：完整流程（推荐）

1. **打开浏览器访问**: http://localhost:3000

2. **上传产品图**
   - 点击上传区域或拖拽图片
   - 建议使用橙子、饮料包装等产品图

3. **等待AI分析** (约5-10秒)
   - 检查提取的品牌、卖点、色彩
   - 如有错误，点击"编辑"修正

4. **选择风格配置**
   ```
   视觉风格: 自然有机 Organic
   排版风格: 粗衬线 Bold Serif
   文字布局: 中英堆叠 Stacked
   画面比例: 9:16 竖版
   ```

5. **点击"下一步"**

6. **进入对比测试**
   - 点击左上角 "🧪 V1 vs V2 对比测试" 按钮

7. **生成对比图像**
   - 点击 "生成图像 (V1)"
   - 等待5-10秒
   - 点击 "生成图像 (V2)"
   - 等待5-10秒

8. **查看对比结果**
   - 左侧：V1生成的图像
   - 右侧：V2生成的图像
   - 从5个维度评分对比

## 🎯 关键观察点

### V2应该更好的地方：

1. **风格一致性**（最重要）⭐⭐⭐
   - 检查是否出现矛盾元素（如自然有机+霓虹灯）
   - V2通过映射表自动统一风格

2. **视觉简洁度**⭐⭐
   - V2只展示1-2个核心卖点
   - 构图更简洁，焦点更集中

3. **产品还原度**⭐⭐
   - V2强化了"Strictly restore"指令
   - 产品细节应该更准确

### 需要注意的地方：

1. **文字质量**
   - 如果V2文字效果差，可以考虑后期Overlay

2. **信息完整性**
   - 如果需要展示所有5个卖点，V1可能更合适

## 🔧 调试技巧

### 如果图像生成失败：

1. **检查API配置**
   ```bash
   cat .env.local | grep GEMINI_API_KEY
   ```

2. **查看浏览器控制台**
   - 按F12打开开发者工具
   - 查看Console和Network标签

3. **检查服务状态**
   ```bash
   # 查看进程
   lsof -ti:3000

   # 查看日志
   tail -f .next/server/app-page.log
   ```

### 如果对比页面打不开：

1. **确认已上传产品图**
   - 必须先完成产品分析和风格选择

2. **检查路由**
   ```bash
   curl http://localhost:3000/compare
   # 应该返回 200
   ```

3. **刷新页面**
   - 清除浏览器缓存
   - 硬刷新: Cmd+Shift+R (Mac) / Ctrl+Shift+R (Windows)
